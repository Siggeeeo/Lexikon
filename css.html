Css och selektorer

Lägga till css kan man göra i en separat fil direkt i sidans head eller direkt på ett element.

Selektorer är det som bestämmer vilka delar av sidan som ska påverkas av css regeln och det kan vara t.e.x alla sidans rubriker, ett specefikt id eller en klass.
