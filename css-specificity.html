Specifikation

#id vinner över .class som vinner över element (exempelvis div). Vid lika så vinner det som står sist i Css-Filen.

Varje regel har ett värde som kan skrivas som x,y,z

  • div { } = (0,0,1)
  • .class { } = (0,1,0)
  • #id { } = (1,0,0)
  • div #id { } = (1,0,1)
  • div#id1 #id { } = (2,0,1)
